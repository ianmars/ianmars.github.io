Final Project: Bidirectional Path Tracing & Multiple Importance Sampling

Summary

We will extend project 3 and implement bidirectional path tracing & multiple importance sampling. We will assume the camera is a pinhole camera.

Problem Description

Normal path tracer converges slowly for caustics and microfacet materials with low alpha, because we are only using paths originated from the camera, which may not model the distribution of radiance from light sources well. Bidirectional path tracer addresses this issue by using paths originated from both the camera and the light sources and connecting any two vertexes across the paths. Unlike normal path tracer, where in one measurement, for each possible length, we only sample at most one path, in bidirectional path tracer, we may sample multiple paths of the same length, then there arises the issue of weighing different paths of the same length before summing them up. To accelerate the convergence of bidirectional path tracer, multiple importance sampling assigns weights to paths according to how likely a path is sampled by the way it is sampled, given the fact that it is sampled (assuming pinhole camera, a path with length k has at most k+1 ways to be sampled). With bidirectional path tracing & multiple importance sampling, we should expect a significant increase in the speed of convergence over normal path tracing, while both methods should ultimately converge to the same image.

Goals and Deliverables

The baseline plan for this project is to implement bidirectional path tracing & multiple importance sampling for a pinhole camera that support any materials and light sources used by any .dae files given in project 3, with the exception of environment light. Our bidirectional path tracing should converge significantly better than normal path tracing with the same amount of samples per pixel or the same amount of time (although for the same amount of samples per pixel, bidirectional path tracing is expected to take more time than normal path tracing). We will show comparisons of both methods to demonstrate the improvement (especially those where normal path tracing performs poorly).

The aspirational plan for this project can be to add support for environment light, to implement photon mapping, and/or to further upgrade bidirectional path tracing into metropolis light transport, which performs better on caustics.
